# test

Sandcastle ile otomatik geliştirme döngüsünü denemek için oluşturulmuş basit bir proje.

## Kurulum

```bash
npm install
```

## Komutlar

| Komut | Açıklama |
|-------|----------|
| `npm run sandcastle` | Sandcastle ajan döngüsünü başlatır |
| `npm run test` | Testleri çalıştırır |
| `npm run typecheck` | TypeScript tip kontrolü yapar |
